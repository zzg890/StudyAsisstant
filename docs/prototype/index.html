一次函数完整笔记

数学 · 初二 · 生成于 2024年1月15日

本节重点

  • 一次函数的定义与一般形式 y = kx + b
  • k 和 b 的几何意义
  • 一次函数的图像与性质

一、一次函数的定义

形如 y = kx + b（k、b 为常数，k ≠ 0）的函数叫做一次函数。

特例：当 b = 0 时，y = kx 叫做正比例函数，是一次函数的特殊形式。

二、k 和 b 的意义

k 的意义

k > 0 时，图像经过一、三象限

k < 0 时，图像经过二、四象限

|k| 越大，直线越陡

b 的意义

b > 0 时，图像与 y 轴交于正半轴

b < 0 时，图像与 y 轴交于负半轴

|b| 越大，截距越大

三、常见考题类型

📝 类型一：求函数解析式

已知一点和斜率，或已知两点坐标，代入求 k、b

📝 类型二：判断图像位置

根据 k、b 的正负判断图像经过的象限

📝 类型三：与坐标轴交点

与 x 轴交点 (-b/k, 0)，与 y 轴交点 (0, b)
